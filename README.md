# 社区医院管理系统 java75

## 项目概述

社区医院管理系统是基于Springboot+Vue的前后端分离项目，采用Javaee技术栈进行开发。本项目遵循MVC三层架构原则，结合Spring、SpringBoot、Mybatis框架以及Vue.js和Maven工具，打造了一套操作便捷、界面简洁的医院管理系统。

## 技术栈

- 前端：Vue.js
- 后端：Springboot、Javaee
- 数据库：MySQL
- 构建工具：Maven

## 功能模块

- 用户管理
- 病患信息管理
- 医生与科室管理
- 药品库存管理
- 就医预约管理

## 系统角色

- 系统管理员
- 医生
- 药剂师
- 病患

## 运行环境

- 服务器：支持Java的应用服务器
- 数据库：MySQL 5.7 或更高版本
- 客户端：现代浏览器

## 部署步骤

1. 搭建Java运行环境
2. 安装并配置MySQL数据库
3. 部署后端服务
4. 部署前端服务
5. 访问系统

## 核心亮点

- 前后端分离架构，便于维护和扩展
- 使用Springboot和Vue，提高了开发效率
